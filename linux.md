# Commands
 * cut
     * [a range of columns](https://stackoverflow.com/questions/4956873/how-to-cut-first-n-and-last-n-columns/51005303)
        > `# returns up to column 2`. 
        > `cut -f -2 $FILE`.
        > `# returns everything after column 2`.   
        > `cut -f 2- $FILE`.
        > `# returns columns 2 to 4`.  
        > `cut -f 2-4 $FILE`.
